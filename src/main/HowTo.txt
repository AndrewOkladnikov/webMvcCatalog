Проект построен на SpringBoot.
Точка запуска - Application -> run.
Стартовый api - http://localhost:8080/
Это главная страница с выводом списка всех продуктов, либо с фильтрацией по категориям продуктов(в выпадающем списке
    выбираем категорию и жмём "Найти")
С главной страницы есть ссылки :
    на категории товаров
    на добавление нового товара.
        При добавлении нового товара обязательным является только название товара. Если не ввести название, то будет соответствующая
            подсказка.
    на редактирование товара(ссылка у названия товара)
    на редактирование категории(ссылка у категории товара товара)
        на этой странице есть кнопка для удаления категории. При удалении категории удаляются все соответствующие товары
На главной странице есть кнопка для удаления товара
При редактировании полей "Описание" и "Название" в товаре или в категории введено ограничение на количество введенных символов.

Тесты написаны для класса PostController. Для сущностей и репозиториев тесты решено было не писать
